# 🚚 Foodtruck Project - [GuttstaFoodtruck]

Detta kommer att bli ett webbprojekt byggt från grunden för en lokal foodtruck. Webbplatsen är optimerad för att kunder snabbt ska kunna hitta dagens position och läsa senaste nytt via sociala medier.

## ✨ Funktioner

* **Live-position:** En interaktiv karta (Google Maps eller Leaflet) som visar var trucken befinner sig just nu.
* **Social Feed:** Integration med Facebook Graph API för att visa de senaste inläggen som en blogg direkt på sidan.
* **Responsiv design:** Anpassad för mobilanvändare.
* **Meny:** En digital meny som är lätt att läsa och navigera.

## 🛠 Teknikstack

* **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox), JavaScript (Vanilla).
* **Integrationer:** * Facebook Graph API (för inlägg).
    * Google Maps API / Mapbox (för lokalisering).
